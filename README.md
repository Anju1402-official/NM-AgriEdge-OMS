# 🌾 AgriEdge OMS - Salesforce Project

## 📌 Project Overview
AgriEdge Or-Mange Ltd is implementing a Salesforce-based Order Management System (OMS) to improve order processing, inventory tracking, and customer service.

## 🚀 Features
- Automated Order Processing
- Real-time Inventory Tracking
- Customer Service Integration
- Reporting & Analytics

## 🛠️ Technologies Used
- Salesforce CRM
- Apex Classes & Triggers
- Process Builder / Flow

## 📂 Project Structure
- apex/ → Apex code (Triggers, Classes)
- screenshots/ → Project demo images
- docs/ → Documentation
